A simple React Pokédex app that lets you search for Pokémon and view their stats, abilities, and images using the [PokéAPI](https://pokeapi.co/).

## Live Demo

Check out the live version: [https://pokewdexxx.netlify.app/](https://pokewdexxx.netlify.app/)

## Features

- Search for any Pokémon by name or ID
- View Pokémon image, types, abilities, and stats
- Responsive and modern UI

## Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```

2. **Start the development server:**
   ```
   npm run dev
   ```

3. **Open in browser:**
   Visit [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

## Project Structure

```
src/
  App.jsx        # Main app component
  Home.jsx       # Home page and logic
  Search.jsx     # Search input component
  Image.jsx      # Pokémon image display
  Stats.jsx      # Stats and abilities display
  App.css        # Main styles
```
